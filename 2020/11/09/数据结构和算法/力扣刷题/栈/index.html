有效的括号

给定一个只包括 (，)，{，}，[，] 的字符串，判断字符串是否有效。

  1. 输入: () 输出: true
  2. 输入: ()[]{} 输出: true
  3. 输入: (] 输出: false
  4. 输入: ([)] 输出: false
  5. 输入: {[]} 输出: true
