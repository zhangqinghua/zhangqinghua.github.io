    常见问题

    1. 登录报错 error 2059: Authentication plugin 'caching_sha2_password' cannot be loaded
      这是因为 MySQL8.0 版本默认的认证方式是 caching_sha2_password。若想 在MySQL8.0 版本中继续使用旧版本中的认证方式需要在 my.cnf 文件中配置并重启，因为此参数不可动态修改。
