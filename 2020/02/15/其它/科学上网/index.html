科学上网

Github

修改hosts：

  1. 打开Dns检测|Dns查询 - 站长工具
  2. 在检测输入栏中输入需要解析的域名；
  3. 把检测列表里的TTL值最小的IP输入到 /etc/hosts 里，并对应写上域名；

在未修改之前，速度在 10kb/s 或者连不上，修改之后，速度可以达到 80kb/s。
