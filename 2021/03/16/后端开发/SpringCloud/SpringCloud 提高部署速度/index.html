打包指定的模块

在多模块的 Maven 项目中，如果每次打包整个工程显得有些冗余和笨重。

我们可以使用命令 mvn clean package install -pl 模块的名称 -am 打包指定模块。

  1. -pl
    打包指定模块。
  2. -am
    打包所指定模块的依赖模块。
