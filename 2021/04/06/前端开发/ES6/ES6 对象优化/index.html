新增的 API

ES6 给 Object 拓展了许多新的方法，如：

  1. keys(obj)
    获取对象的所有 key 形成的数组。
  2. values(obj)
    获取对象的所有 value 形成的数组。
  3. entries(obj)
    获取对象的所有 key 和 value 形成的二维数组。格式：[[k1,v1],[k2,v2],...]。
  4. assign(dest, ...src)
    将多个 src 对象的值 拷贝到 dest 中。（第一层为深拷贝，第二层为浅拷贝）。
