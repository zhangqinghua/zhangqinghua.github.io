修改容器启动配置参数

有时候，我们创建容器时忘了添加参数 --restart=always，当 Docker 重启时，容器未能自动启动。现在要添加该参数怎么办呢，方法有二：

1. Docker 命令修改

2. 配置文件修改

  1. 首先停止容器，不然无法修改配置文件
  2. 配置文件路径为：/var/lib/docker/containers/容器ID
  3. 在该目录下找到一个文件 hostconfig.json ，找到该文件中关键字 RestartPolicy
  4. 修改前配置：”RestartPolicy”:{“Name”:”no”,”MaximumRetryCount”:0}
  5. 修改后配置：”RestartPolicy”:{“Name”:”always”,”MaximumRetryCount”:0}
  6. 最后启动容器。
