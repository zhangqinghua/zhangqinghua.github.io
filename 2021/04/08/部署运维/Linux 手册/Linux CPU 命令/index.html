vmstat 查看 CPU 概要

vmstat 用户查看 CPU 信息（包括但不限于），一般 vmstat 的使用是通过两个数字参数来完成，第一个参数是采样的时间间隔（s），第二个参数是采样的次数。

mpstat 查看 CPU 明细

mpstat 用于查看所有的CPU信息。

pidstat 查看进程的 CPU 占用

pidstat 用于查看每个进程使用 CPU 和内存的用量分解信息。
