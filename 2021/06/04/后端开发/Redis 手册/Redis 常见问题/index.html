MISCONF Redis is configured to save RDB snapshots, but it is currently not able to…

场景：Docker 部署 Redis，隔几周就报这个异常。
原因：Redis 快照关闭了导致不能持久化。
解决：设置 stop-writes-on-bgsave-error: no。
